# Figma Import Notes

This folder contains `figma-import.svg`, a Figma-importable visual mockup of the wedding website.

## Import

1. Open Figma.
2. Drag `figma-import.svg` onto the canvas.
3. If needed, replace text layers manually with editable Figma text.

## Fonts

- Headlines: `moret`
- Labels / sub-headlines: `ballinger-mono`
- Body text: `Inter`

Both `moret` and `ballinger-mono` come from:

```html
<link rel="stylesheet" href="https://use.typekit.net/eot6kmk.css">
```

## Colors

- Blush: `#f3dcd7`
- Light blush: `#f8e8e5`
- Red: `#ff241f`
- Dark red/brown: `#4b140f`
